# primes
Find prime numbers

## Usage

```Bash
$ ./primes --help
Usage:
  primes --before <NUMBER> [--benchmark]
  primes --help
```


## Dependencies

- Linux
- Node 4.2.4+


## Examples

```Bash
$ ./primes --before 10
2
3
5
7
```

```Bash
$ ./primes --before 20 --benchmark
2
3
5
7
11
13
17
19
Time elapsed: 4ms
```
